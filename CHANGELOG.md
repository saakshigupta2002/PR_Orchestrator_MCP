## [0.1.0] - 2026-01-27
### Added
* Initial release of **PR Orchestrator MCP**.  Includes a fully scaffolded repository with a server that exposes workspace, repository, editing, QA, GitHub and approval tools over stdio.  Implements safety policies, configuration loading, branch strategy, artifact bundling and telemetry.  Provides comprehensive documentation, example environment configuration and smoke tests.
